## 3. Generator (the no-embed keystone)

- [x] 3.1 `compileSecretsModule` emits `home.file.<homeRelTarget>.source =
      config.lib.file.mkOutOfStoreSymlink <path>;` for a `path` entry (out-of-store symlink at
      activation — no store import, pure-eval-safe), sorted by name.
- [x] 3.2 Stage the result as `./secrets.nix` and add it to the generated flake's `modules` list;
      thread secrets through `writeHomeFlake` and both public generators (settings + config).
- [x] 3.3 STRUCTURAL: a secret entry never enters the `staged` map and is never `os.ReadFile`'d;
      a sentinel at a secret path is absent from every generated byte.
